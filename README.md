# Smart Context-Aware To-Do List

## Description
This project is a smart task management system that recommends tasks based on user context, including energy level, available time, deadline urgency, and location.

## Features
- Add, edit, and delete tasks
- Task recommendation system
- Context-aware filtering (home/outside)
- Task search
- Statistics visualization

## Technologies Used
- Python
- Streamlit
- Matplotlib
- JSON

## How to Run
1. Install dependencies:
   pip install streamlit matplotlib

2. Run the app:
   streamlit run app.py
